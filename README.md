# ECommerce_Logistics_dashboard
Interactive Excel dashboard analysing ecommerce logistics KPIs — delivery performance, return rates, and shipping cost variances across courier partners.
## Ecommerce Logistics Dashboard

**Tools:** Advanced Excel | Power Query | Pivot Tables | Dynamic Charts

### Overview
An interactive Excel dashboard built to analyse ecommerce logistics 
performance, identify delivery inefficiencies, and present key operational 
KPIs to management for data-driven decision-making.

### Key Features
- KPI tracking: On-Time Delivery %, Return Rate, Cost Per Shipment
- Delivery delay analysis across courier partners
- Return rate and shipping cost variance identification
- Data cleaned and validated using Power Query
- Interactive Pivot Tables and dynamic charts
- Systematic data validation and reconciliation at every stage

### Insights Delivered
- Identified patterns in delivery delays across courier partners
- Highlighted high return rate segments for operational review
- Surfaced shipping cost variances to support cost optimisation decisions

### How to View
1. Download the `.xlsx` file
2. Open in **Microsoft Excel** (2016 or later recommended)
3. Enable editing and macros if prompted
